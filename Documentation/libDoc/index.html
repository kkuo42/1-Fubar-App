validPassword

Calls compareSync to check the given password against the password stored in the database

validPassword(storedPassword: String, givenPassword: String): boolean
Parameters
storedPassword (String) The password stored in the database
givenPassword (String) The password given by the user
Returns
boolean: True if the passwords matched, false otherwise

validateLogin

Verifies that a login is valid before calling a function

validateLogin(db: Object, email: String, password: String, fn: Function): any
Parameters
db (Object) A mongoDB
email (String) The email used to login
password (String) The password used to login
fn (Function) The callback function
Returns
any: fn(true) if the login is successful, fn(false) otherwise

isValidToken

Verifies if a mobile token is proper.

isValidToken(db: Object, authHeader: Object, fn: Function): any
Parameters
db (Object) A mongoDB
authHeader (Object) The req.headers.authorization token. Expect to be in form Token _id
fn (Function) Callback function
Returns
any: fn(false) if an error was found, fn(true) otherwise

isAuthenticated

Middleware authentication which will call the next function if successful otherwise 403 forbidden

isAuthenticated(req: Object, res: Object, next: Function): any
Parameters
req (Object) The request object
res (Object) The response object
next (Function) The function to call if successful
Returns
any: next() if req contains a valid token, 403 otherwise
